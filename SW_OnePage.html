JVK SW Check-In System
Web App บันทึกการปฏิบัติงานทุก Ward แบบ Real-time พร้อมระบบ Admin อนุมัติและส่งอีเมลแจ้งผลอัตโนมัติ

  • บันทึก Ward · กิจกรรม · ผู้ป่วย ทันที
  • Admin เห็น Pending → Approve ได้ทันที
  • แจ้งผลทาง Email อัตโนมัติทุก Visit
  • Dashboard: Active / Waiting / Approved
